## 🚀 Getting Started

### Prerequisites
```bash
node >= 18.0.0
python >= 3.11
postgresql >= 15
redis >= 7.0
```

### Installation

```bash
# Clone the repository
git clone https://github.com/yourusername/phantom-mind.git
cd phantom-mind

# Install frontend dependencies
cd frontend
npm install

# Install backend dependencies
cd ../backend
npm install

# Install Python ML dependencies
cd ../ml
pip install -r requirements.txt

# Set up environment variables
cp .env.example .env
# Fill in your API keys (see Environment Variables section)
```

### Environment Variables
```env
# Anthropic
ANTHROPIC_API_KEY=your_key_here

# Market Data
BINANCE_API_KEY=your_key_here
BINANCE_SECRET=your_secret_here
ALPACA_API_KEY=your_key_here
POLYGON_API_KEY=your_key_here

# On-Chain
ETHERSCAN_API_KEY=your_key_here
GLASSNODE_API_KEY=your_key_here

# News
NEWS_API_KEY=your_key_here

# Database
DATABASE_URL=postgresql://localhost:5432/phantom_mind
REDIS_URL=redis://localhost:6379
```

### Running the App

```bash
# Start backend server
cd backend && npm run dev

# Start ML pipeline
cd ml && python pipeline.py

# Start frontend
cd frontend && npm start

# Dashboard available at http://localhost:3000
```

## 🤖 AI & ML Features

### Phantom AI Oracle
The conversational AI is powered by Anthropic's Claude. The system prompt is dynamically injected with live market context on every query — consciousness score, whale activity, narrative health, options flow — making every response genuinely current and actionable.

### FinBERT News Sentiment
Financial news articles are classified using a fine-tuned version of BERT trained on financial text. Each article receives a directional score (-1 to +1) and an impact weight based on source credibility and historical price-moving power.

### Behavioral Genome Engine
Each asset's psychological profile is computed from 180 days of historical data across 6 dimensions: fear sensitivity, hype cycle duration, Reddit-to-price lag, manipulation vulnerability, recovery resilience, and institutional vs. retail ratio.

### Narrative Collapse Detection
NLP tracks the semantic density and media saturation of active market narratives. When a narrative exceeds 85% saturation (measured by unique source coverage and sentiment uniformity), the decay signal fires — historically preceding major sector rotations by 2–4 weeks.

### Temporal Fusion Transformer
A state-of-the-art time-series model that incorporates price history, sentiment signals, volume anomalies, and macro indicators to generate the AI prediction zone displayed on every chart.
